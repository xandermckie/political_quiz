# Marx Values

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

A fun, educational quiz that helps people explore their place on the socialist spectrum — from orthodox Marxism to anarchism and beyond. Built with plain HTML, CSS, and JavaScript.

**Live site:** [marxvalues.org](https://marxvalues.org)

## 📋 About

**Marx Values** is an interactive political quiz designed to make socialist ideas more accessible. Instead of a simple left-right axis, it maps users across tendencies within socialist thought, helping them discover traditions they might want to explore further.

The goal is to combine fun with education: after the quiz, users receive curated resources — books, articles, videos, and organizations — to begin or deepen their own study of Marxist and socialist ideas.

## ✨ Features

- **Interactive quiz** – A series of questions that place users on a multi-axis ideological map
- **Detailed results** – Explains the user's alignment and provides a starting point for further learning
- **Resources page** – A curated collection of introductory and advanced materials
- **Mobile-friendly** – Responsive design works on phones, tablets, and desktops
- **No tracking** – The quiz runs entirely in the browser; no data is collected or stored

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Hosted via Cloudflare Pages

## 🚀 Getting Started Locally

To run the site on your own machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/marxvalues.git
   cd marxvalues
   ```

2. **Open locally** – just open any `.html` file in your browser, or use a simple local server:
   ```bash
   # Python 3
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

No build steps, dependencies, or installation required.
